LangChain的流式输出
```python

agent = create_agent(  
    model=model,  
    tools=[get_weather],  
    checkpointer=InMemorySaver()  
)  
config: RunnableConfig = {'configurable': {'thread_id': '001'}}  
# 返回一个流模式迭代器  
input_content = {'messages': [HumanMessage('贵阳天气怎么样啊,适合外出吗?')]}  
stream_iterator = agent.stream(input_content, config=config, stream_mode=['messages'],version='v1或v2,v3')
for chunk in stream_iterator:
	1.当version=v1,默认v1
		stream_mode=['messages']
			return: ("messages", (AIMessageChunk, meta))
		stream_mode='messages'
			return: (AIMessageChunk, metadata)
		stream_mode=['messages','custom']
			return: ("messages", ...) 和 ("custom", ...) 交替 yield
	2.当version=v2 stream_mode=任意
		return: {'type': mode,'ns':(), 'data': payload}
	3.当version=v3,不适用.
		agent.stream() 不支持 v3, version 只接受 'v1' | 'v2'
		v3 只能用来调用 agent.stream_events(version='v3')，返回的是 GraphRunStream 对象
	
	注意: v1和v2的取值方式不同,需要自己进行条件判断
	
```

> [!caution]  警告
> 为了统一API，建议在使用流式输出的时候都统一使用 version='v2'
