    tomcat部署war包方式安装jenkins 1.首次登陆忘记密码 修改 /root/.jenkins/config.xml 去除配置 true true 登陆jenkis创建用户 重启Jenkins服务（如果是war包方式的部署，即重启tomcat）； 进入首页>“系统管理”>“Configure Global Security”； 勾选“启用安全”； 点选“Jenkins专有用户数据库”，并点击“保存”； 重新点击首页>“系统管理”,发现此时出现“管理用户”； 点击进入展示“用户列表”； 点击右侧进入修改密码页面，修改后即可重新登录
